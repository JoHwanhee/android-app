조건문 예제



두 수 (a b) 를 비교하고 더 큰 값을 result라는 변수에 담아보도록 합시다.



```
#include <stdio.h>
 
int main ( void ) {
 
    int x;
    int y;
    int result;
 
    x = 10;
    y = 8;
 
    if ( x > y ) {
        result = x;
    }
 
    if ( y > x ) {        
        result = y;
    }
 
    return 0;
}
```


이런식으로 if문을 코드에 적용할 수 있습니다!!
